Armstrong Function

This function checks whether the input number is an Armstrong number or not.

function isArmstrong(number) {
    let sum = 0;
    let temp = number;
    const length = number.toString().length;
    while (temp > 0) {
        let digit = temp % 10;
        sum += Math.pow(digit, length);
        temp = Math.floor(temp / 10);
    }
    return sum === number;
}

Parameters:

  • Name: number
  • Type: Number
  • Description: The input number to apply the check on (can be of any length).

Returns:

  • Type: Boolean
  • Description: If the input is an Armstrong number, it will return true; otherwise, false.
